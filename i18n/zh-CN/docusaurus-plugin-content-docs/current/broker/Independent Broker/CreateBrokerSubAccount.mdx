**类型:** POST

**描述:** `/v4/user/account/broker-sub`

在经纪商母账户下创建子账户。请求必须使用**经纪商母账户**的 API Key 签名。子账户与母账户共享 `userId`，并拥有独立的 `accountId`。

### 参数

| 名称        | 类型   | 是否必填 | 默认值 | 描述                                 | 取值范围 |
| ----------- | ------ | -------- | ------ | ------------------------------------ | -------- |
| accountName | string | 是       | -      | 子账户显示名称，仅限字母和数字       | 6-16     |

---

### 响应参数

| 名称         | 类型 | 描述               |
| ------------ | ---- | ------------------ |
| subAccountId | long | 新创建的子账户 id  |

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": {
    "subAccountId": 10049631158745
  },
  "success": true
}
```

---

### 错误码

| 错误码                     | 说明                                          |
| -------------------------- | --------------------------------------------- |
| USER_SUB_ACCOUNT_ILLEGAL   | `accountName` 不符合规则（字母/数字，6-16 位） |
| SUB_ACCOUNT_LIMIT          | 该母账户的子账户数量已达上限                  |
