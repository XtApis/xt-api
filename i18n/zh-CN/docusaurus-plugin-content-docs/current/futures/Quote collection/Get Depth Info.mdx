### 请求方式
- 类型: **GET**
- 地址: `/future/market/v1/public/cg/orderbook`
- 注意: 该方法不需要签名

---


### 请求参数

| 名称 | 类型 | 是否必需 | 默认值 | 描述 | 范围 |
|------|------|-----------|--------|------|------|
| symbol | string | TRUE | N/A | 交易对 |  |
| level | int | FALSE | 50 | 深度档位 | 1-200 |

---

### 请求示例

```bash
curl -G "https://fapi.xt.com/future/market/v1/public/cg/orderbook" \
    -H "Content-Type: application/x-www-form-urlencoded" \  # 必须
    -d "symbol=eth_usdt"  # 必须
```

---

### 响应示例

```json
{
    "ticker_id": "BTC-USDT",
    "timestamp": 1698668957638,
    "bids": [
      ["34794.6", "97164"],
      ["34794.5", "9897"],
      ...
    ],
    "asks": [
      ["34794.7", "168479"],
      ["34794.8", "4009"],
      ...
    ]
}
```

---

### 错误码

| Error code | Desc |
|-------------|------|
| invalid_symbol | 交易对不存在 |
