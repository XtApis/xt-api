**类型:** DELETE

**描述:** `/v4/user/account/api-key/broker-sub`

删除子账户的 API Key。请求必须使用**经纪商母账户**的 API Key 签名。

### 参数

| 名称      | 类型   | 是否必填 | 默认值 | 描述                                             | 取值范围 |
| --------- | ------ | -------- | ------ | ------------------------------------------------ | -------- |
| accessKey | string | 是       | -      | 要删除的 key（必须属于该经纪商名下的子账户）     | -        |

---

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": true,
  "success": true
}
```
