**请求类型** DELETE

**接口描述：** `/v4/open-order`

---

### 请求参数

| 参数名   | 类型   | 是否必填 | 默认值 | 描述                                   | 取值范围     |
|----------|--------|----------|--------|----------------------------------------|--------------|
| symbol   | string | 否       | N/A    | 交易对，不填则表示全部                  | —            |
| bizType  | string | 是       | N/A    | 业务类型                                | SPOT, LEVER  |
| side     | string | 否       | N/A    | 订单方向                                | BUY, SELL    |

---

### 限频规则
- 10次/秒/每apikey
- **注意：** 参数需以 **JSON** 格式放置于请求体中。

### 请求示例

```
curl --location --request DELETE 'https://sapi.xt.com/v4/open-order' \
--header 'accept: */*' \
--header 'Content-Type: application/json' \
--header 'validate-algorithms: HmacSHA256' \
--header 'validate-recvwindow: 60000' \
--header 'validate-appkey: xxxxxxxxxx' \
--header 'validate-timestamp: xxxxxxxxxx' \
--header 'validate-signature: xxxxxxxxxx'
--data '{
    "bizType": "SPOT",
    "symbol": "XT_USDT"
}'
```

---

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "string",
  "ma": [{}],
  "result": {}
}
