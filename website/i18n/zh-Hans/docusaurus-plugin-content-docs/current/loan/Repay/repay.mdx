### 描述
**接口地址：**
`POST /v4/order/finance/loan/order/repay`

---

### 请求参数

<CodeBlock language="json" title="请求参数">
{`{
  "positionId": 12345,                 // 持仓ID
  "amount": "15000",                   // 还款金额
  "pledgeRatio": "true",               // 是否按还款比例释放质押物
  "bizId": "repay_123456789"           // 业务ID
}`}
</CodeBlock>
