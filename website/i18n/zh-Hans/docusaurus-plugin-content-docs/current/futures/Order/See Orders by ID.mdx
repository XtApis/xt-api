**类型:** GET

**描述:** `/future/trade/v1/order/detail`

### 参数

| 名称    | 类型    | 必填 | 默认值 | 描述 | 范围 |
| ------- | ------- | --------- | ------- | ----------- | ------ |
| orderId | integer | true      | N/A     | 订单 ID    |        |

### 限流规则

200/s/apikey

### 请求示例

<CodeBlock language="bash" title="Request">
{`curl -G "https://fapi.xt.com/future/trade/v1/order/detail" \\
  -H "validate-appkey: $APPKEY" \\  # 必须
  -H "validate-timestamp: $TIMESTAMP" \\  # 必须
  -H "validate-signature: $SINGATURE" \\  # 必须
  -H "Content-Type: application/x-www-form-urlencoded" \\  # 必须
  -d "orderId=1234567890"  # 必须
`}
</CodeBlock>

### 错误码

| 错误码 | 描述 |
| --- | --- |
| null | 未找到该订单 |
