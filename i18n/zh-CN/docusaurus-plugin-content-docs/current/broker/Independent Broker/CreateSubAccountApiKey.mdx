**类型:** POST

**描述:** `/v4/user/account/api-key/broker-sub`

为经纪商子账户创建 API Key。请求必须使用**经纪商母账户**的 API Key 签名。**不允许**授予 `WITHDRAW`（提现）权限。

### 参数

| 名称         | 类型     | 是否必填 | 默认值 | 描述                                         | 取值范围                       |
| ------------ | -------- | -------- | ------ | -------------------------------------------- | ------------------------------ |
| subAccountId | long     | 是       | -      | 目标子账户 id（必须属于当前母账户）          | -                              |
| keyName      | string   | 是       | -      | 备注 / 标签                                  | -                              |
| permissions  | string[] | 是       | -      | 权限码；`WITHDRAW` 会被拒绝                  | QUERY, TRADE, TRANSFER, FUTURE |
| bindIps      | string[] | 否       | -      | IP 白名单                                    | -                              |

---

### 响应参数

:::caution
`secretKey` **仅在创建时返回一次**，请立即妥善保存，之后无法再次获取。
:::

| 名称         | 类型     | 描述                             |
| ------------ | -------- | -------------------------------- |
| subAccountId | long     | 子账户 id                        |
| accessKey    | string   | 新创建的 access key              |
| secretKey    | string   | secret key —— 仅返回一次，请保存 |
| permissions  | string[] | 已授予的权限                     |
| bindIps      | string[] | 已绑定的 IP                      |
| createTime   | long     | 创建时间（毫秒）                 |

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": {
    "subAccountId": 10049631158745,
    "accessKey": "69af55c4-e26c-4cea-a0b4-61a33c400840",

    "permissions": ["QUERY"],
    "bindIps": [],
    "createTime": 1784088472976
  },
  "success": true
}
```
