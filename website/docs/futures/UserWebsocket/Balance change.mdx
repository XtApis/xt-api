### Subscribe / Unsubscribe

```json title="Subscribe / Unsubscribe"
{
  "method": "SUBSCRIBE/UNSUBSCRIBE",
  "params": ["balance@{ListenKey}"],
  "id": "{id}"
}
```

### Response

```json title="Response example"
{
  "topic": "balance",
  "event": "balance@123456",
  "data": {
    "coin": "usdt",
    "underlyingType": 1, // 1: coin‑margined; 2: USDT‑margined
    "walletBalance": "123", // balance
    "openOrderMarginFrozen": "123", // frozen open orders
    "isolatedMargin": "213", // isolated margin
    "crossedMargin": "0" // crossed margin
  }
}
```
