**请求类型:** GET
**Description:** `/future/trade/v1/entrust/plan-list-history`

---

### 限流规则
200/s/apikey

---

### 参数说明
| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
|------|------|------|--------|------|------|
| symbol | string | TRUE | N/A | 交易对（不传则查询所有交易对） | — |
| direction | string | FALSE | NEXT | 查询方向 | PREV; NEXT |
| id | integer | FALSE | N/A | ID | — |
| limit | integer | FALSE | 10 | 数量限制 | — |
| startTime | integer | FALSE | N/A | 开始时间 | — |
| endTime | integer | FALSE | N/A | 结束时间 | — |

---

### 请求示例
```bash
curl -G "https://fapi.xt.com/future/trade/v1/entrust/plan-list-history" \
    -H "validate-appkey: $APPKEY"              # 必填
    -H "validate-timestamp: $TIMESTAMP"        # 必填
    -H "validate-signature: $SIGNATURE"        # 必填
    -H "Content-Type: application/x-www-form-urlencoded"  # 必填
    -d "symbol=btc_usdt"                       # 必填
    -d "direction=NEXT" \
    -d "limit=10"
```
