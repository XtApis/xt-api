**类型：** Websocket 推送  
**描述：** 余额变动通知

### 参数

| 名称 | 类型 | 是否必须 | 默认值 | 描述 |
| --- | --- | --- | --- | --- |
| – | – | false | N/A | 无需参数，订阅 **balance** 主题即可接收更新 |

### 说明

- 当用户账户余额发生变化时触发此推送通知。
- 订阅格式：`balance`
- 支持 **SPOT** 和 **LEVER** 两种业务类型。
- 可用于实时跟踪交易、充值、提现或转账后的余额变化。

### 推送示例

```json title="推送"
{
  "topic": "balance",
  "event": "balance",
  "data": {
    "a": "123", // 账户 ID
    "t": 1656043204763, // 事件时间（毫秒）
    "c": "btc", // 币种
    "b": "123", // 现货总余额
    "f": "11", // 冻结金额
    "z": "SPOT", // 业务类型 [SPOT, LEVER]
    "s": "btc_usdt" // 交易对
  }
}
```
