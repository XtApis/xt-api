**类型:** GET  
**描述:** `/future/trade/v1/position/list/active`

### 限流规则
`200/s/apikey`

### 请求参数
| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
|------|------|------|--------|------|------|
| symbol | string | FALSE | N/A | 交易对 |  |
| isDelivery | bool | FALSE | FALSE | 是否查询交割 |  |

### 请求示例
```bash
curl -G "https://fapi.xt.com/future/trade/v1/position/list/active" \
  -H "validate-appkey: $APPKEY" \  # 必须
  -H "validate-timestamp: $TIMESTAMP" \  # 必须
  -H "validate-singature: $SIGNATURE" \  # 必须
  -H "Content-Type: application/x-www-form-urlencoded" \  # 必须
  -d "symbol=btc_usdt" \ 
```
