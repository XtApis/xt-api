**请求方式：** POST


**接口描述：** `/future/trade/v1/entrust/cancel-all-plan`


---


### 限流规则
- 200 次/秒/每个 apikey


---


### 请求参数


| 参数名 | 类型 | 是否必填 | 默认值 | 参数说明 | 取值范围 |
|--------|--------|-----------|---------|-----------|-----------|
| symbol | string | 是 | — | 交易对 | 例如 btc_usdt |


---


### 请求示例


```bash title="cURL"
curl -X POST "https://fapi.xt.com/future/trade/v1/entrust/cancel-all-plan" \
-H "validate-appkey: $APPKEY" \                           # 必填
-H "validate-timestamp: $TIMESTAMP" \                     # 必填
-H "validate-signature: $SIGNATURE" \                     # 必填
-H "Content-Type: application/x-www-form-urlencoded" \    # 必填
-d "symbol=btc_usdt"                                      # 必填
```


---


### 返回示例


```json title="Response"
{
  "error": {
  "code": "",
  "msg": ""
},
  "msgInfo": "",
  "result": {},
  "returnCode": 0
}
```

---


### 错误码说明


| 错误码 | 描述 |
|------------------|----------------|
| invalid_symbol | 交易对不存在 |
| fail | 内部错误 |
