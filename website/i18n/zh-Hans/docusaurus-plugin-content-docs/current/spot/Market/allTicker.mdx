### 接口说明
**GET** `/v4/public/ticker`

---

### 限流规则
1. 单一交易对：10次/秒/ip
2. 多个交易对：10次/秒/ip

---

### 请求参数

| 参数名   | 类型   | 是否必填 | 默认值 | 描述                                        | 取值范围 |
|----------|--------|----------|--------|---------------------------------------------|----------|
| symbol   | string | 否       | N/A    | 交易对名称，例如：`btc_usdt`                | -        |
| symbols  | array  | 否       | N/A    | 交易对集合，优先级高于 symbol，例如：`btc_usdt,eth_usdt` | - |
| tags     | string | 否       | N/A    | 标签集合，逗号分隔，目前仅支持 `spot`        | -        |

---
