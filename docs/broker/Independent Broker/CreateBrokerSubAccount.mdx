**Type:** POST

**Description:** `/v4/user/account`

### Parameters

| Name        | Type    | Mandatory | Default | Description                                              | Ranges |
| ----------- | ------- | --------- | ------- | -------------------------------------------------------- | ------ |
| accountName | string  | true      | -       | Sub-account name                                         | 6-16   |
| type        | integer | false     | 1       | Account type: 1-General account, 7-Broker sub-account    | 1, 7   |

---

### Response Example

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": true //true: success; false/null: unsuccess
}
```
