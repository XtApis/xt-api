**请求类型:** POST
**请求路径:** `/v4/order`

---

### 参数说明

| 名称          | 类型   | 必填 | 默认值 | 描述                                                                 | 取值范围                 |
|---------------|--------|------|--------|----------------------------------------------------------------------|--------------------------|
| symbol        | string | 是   | —      | 交易对                                                               | —                        |
| clientOrderId | string | 否   | —      | 自定义订单号，正则: `^[a-zA-Z0-9_]{4,22}$`                          | —                        |
| side          | string | 是   | —      | 订单方向                                                             | BUY, SELL                |
| type          | string | 是   | —      | 订单类型                                                             | LIMIT, MARKET            |
| timeInForce   | string | 是   | —      | 有效方式                                                             | GTC, FOK, IOC, GTX       |
| bizType       | string | 是   | —      | 业务类型                                                             | SPOT, LEVER              |
| price         | number | 否   | —      | 价格。当为 LIMIT 订单时必填；MARKET 订单不需要填写                   | —                        |
| quantity      | number | 否   | —      | 数量。当为 LIMIT 订单或按数量市价单时必填                           | —                        |
| quoteQty      | number | 否   | —      | 金额。当为 LIMIT 订单或按金额市价单时必填                           | —                        |
| nftId         | string | 否   | —      | NFT ID                                                              | —                        |

---

### 备注
- 创建 **买入市价单** 时，`quantity` 必须为空，`quoteQty` 必填。
- 创建 **卖出市价单** 时，`quoteQty` 必须为空，`quantity` 必填。

---
