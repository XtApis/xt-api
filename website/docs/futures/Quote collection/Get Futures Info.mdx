### Request Method

- Type: **GET**
- Description: `/future/market/v1/public/cg/contracts`
- Note: This method does not require signature.

---

### Request Example

```bash
curl -G "https://fapi.xt.com/future/market/v1/public/cg/contracts" \
    -H "Content-Type: application/x-www-form-urlencoded" \  # Required
```

---
