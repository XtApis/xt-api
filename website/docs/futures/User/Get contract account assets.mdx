**Type** GET

**Description:** /future/user/v1/compat/balance/list

#### **Limit Flow Rules**

200/s/apikey

### Parameters

| Name           | Type   | Mandatory | Default | Description |
| -------------- | ------ | --------- | ------- | ----------- |
| queryAccountId | string | false     | N/A     | Account ID  |
