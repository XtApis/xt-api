## 2025-11-04
### 内容新增
  * 合约跟单，新增以下接口
    * /v1/copy-trade/current-following-v2 : 获取用户当前跟随的交易员列表
    * /v1/copy-trade/follower-position : 获取跟单者当前跟单持仓
    * /v1/copy-trade/follower-position-history: 获取跟单者历史跟单持仓
    * /v1/copy-trade/follower-balance-bill:  获取跟单者资金流水
