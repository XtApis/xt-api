**请求类型** POST

**接口描述：** `/future/user/v1/position/adjust-leverage`

**请求参数**

| 参数名       | 类型     | 是否必填 | 默认值 | 描述                 |
| ------------ | -------- | -------- | ------ | -------------------- |
| symbol       | string   | 是       | N/A    | 交易对               |
| positionSide | string   | 是       | N/A    | 持仓方向 (LONG, SHORT) |
| leverage     | integer  | 是       | N/A    | 杠杆倍数             |

### 限流规则

200次/秒/apikey

### 请求示例

```bash title="Request"
curl -X POST "https://fapi.xt.com/future/user/v1/position/adjust-leverage" \
  -H "validate-appkey: $APPKEY" \  # 必须
  -H "validate-timestamp: $TIMESTAMP" \  # 必须
  -H "validate-singature: $SINGATURE" \  # 必须
  -H "Content-Type: application/x-www-form-urlencoded" \  # 必须
  -d "symbol=btc_usdt" \  # 必须
  -d "positionSide=SHORT" \  # 必须
  -d "leverage=20" \  # 必须
```

### 响应示例

```json title="Response"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {},
  "returnCode": 0
}
```

### 错误码

| Error code | Desc |
|-------------|------|
| invalid_symbol | 交易对不存在或被置空了 |
| invalid_leverage | 杠杆倍数不合法 |
