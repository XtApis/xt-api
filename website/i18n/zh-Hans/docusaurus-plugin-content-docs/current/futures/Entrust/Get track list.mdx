**请求类型:** GET
**描述:** `/future/trade/v1/entrust/track-list`

---

### 限流规则

200/s/apikey

---

### 参数说明
| 名称 | 类型 | 必填 | 默认值 | 描述 |
|------|------|------|--------|------|
| page | integer | FALSE | 1 | 页码 |
| size | integer | FALSE | 10 | 每页数量 |
| endTime | integer | FALSE | N/A | 结束时间 |
| startTime | integer | FALSE | N/A | 开始时间 |
| symbol | string | FALSE | N/A | 交易对 |

---

### 请求示例
```bash
curl -G "https://fapi.xt.com/future/trade/v1/entrust/track-list" \
    -H "validate-appkey: $APPKEY"              # 必须
    -H "validate-timestamp: $TIMESTAMP"        # 必须
    -H "validate-signature: $SIGNATURE"        # 必须
    -H "Content-Type: application/x-www-form-urlencoded"  # 必须
    -d "page=1" \
    -d "size=10" \
    -d "symbol=btc_usdt"
```
