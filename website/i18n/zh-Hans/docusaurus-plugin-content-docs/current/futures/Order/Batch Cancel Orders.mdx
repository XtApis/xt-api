**类型:** POST

**描述:** `/future/trade/v1/order/cancel-batch`

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| orderIds | string | true | N/A | 订单 ID（逗号分隔） |  |

### 限流规则

200/s/apikey

### 请求示例

<CodeBlock language="bash" title="Request">
{`curl -X POST "https://fapi.xt.com/future/trade/v1/order/cancel-batch" \
  -H "validate-appkey: $APPKEY" \  # 必须
  -H "validate-timestamp: $TIMESTAMP" \  # 必须
  -H "validate-singature: $SINGATURE" \  # 必须
  -H "Content-Type: application/x-www-form-urlencoded" \  # 必须
  -d "orderIds=111111,222222,333333"  # 必须
`}
</CodeBlock>
