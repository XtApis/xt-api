**类型:** GET  
**描述:** `/future/trade/v1/position/cross-margin/{symbol}`

---
**限流规则**  
200/s/apikey

---
**请求参数**  
-

---
**请求示例**
```bash
curl -G "https://fapi.xt.com/future/trade/v1/position/cross-margin/eth_usdt" \
  -H "validate-appkey: $APPKEY" \  # 必须
  -H "validate-timestamp: $TIMESTAMP" \  # 必须
  -H "validate-singature: $SIGNATURE" \  # 必须
  -H "Content-Type: application/x-www-form-urlencoded" \  # 必须
```

---
**响应示例**
```json
{
    "error": {
      "code": "",
      "msg": ""
    },
    "msgInfo": "",
    "result": "result",
    "returnCode": 0
}
```
