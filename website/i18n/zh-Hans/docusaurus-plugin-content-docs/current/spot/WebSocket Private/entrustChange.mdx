**类型：** Websocket 推送 **描述：** 委托（触发订单）变动通知

### 参数

| 名称 | 类型 | 是否必须 | 默认值 | 描述 |
| --- | --- | --- | --- | --- |
| – | – | false | N/A | 无需参数，订阅 **trigger** 主题即可接收更新 |

### 说明

- 当**触发/委托订单**发生变化时触发此推送通知。
- 订阅格式：`trigger`
- 可用于跟踪触发订单的创建、更新和状态变化。

### 订阅示例

```json title="订阅"
{
  "method": "subscribe",
  "params": ["trigger"],
  "listenKey": "512312356123123123" // 通过 /v4/ws-token 获取的监听密钥
}
```

### 推送示例

```json title="推送"
{
  "topic": "trigger",
  "event": "trigger",
  "data": {
    "s": "btc_usdt", // 交易对
    "t": 1656043204763, // 事件时间（毫秒）
    "i": "6216559590087220004", // 触发订单 ID
    "st": "NEW" // 状态（如 NEW、FILLED、CANCELED）
  }
}
```
