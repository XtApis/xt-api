**类型:** GET

**描述:** `/v4/trade`

### 参数

| 名称 | 类型 | 是否必需 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | string | 否 | N/A | 交易对，如不填写则代表所有 |  |
| bizType | string | 否 | N/A | 业务类型：`SPOT`、`LEVER` |  |
| orderSide | string | 否 | N/A | 订单方向：`BUY`、`SELL` |  |
| orderType | string | 否 | N/A | 订单类型：`LIMIT`、`MARKET` |  |
| orderId | number | 否 | N/A | 订单ID |  |
| fromId | number | 否 | N/A | 起始ID |  |
| direction | string | 否 | N/A | 查询方向：`PREV`、`NEXT` |  |
| limit | number | 否 | 20 | 限制数量，最大100 |  |
| startTime | number | 否 | N/A | 开始时间（例如：`1657682804112`） |  |
| endTime | number | 否 | N/A | 结束时间 |  |

### 请求示例

```bash title="cURL"
curl --location --request GET 'https://sapi.xt.com/v4/trade?limit=100&bizType=SPOT&startTime=1757779200000&endTime=1760457599999&direction=NEXT' \
--header 'accept: */*' \
--header 'Content-Type: application/json' \
--header 'validate-algorithms: HmacSHA256' \
--header 'validate-recvwindow: 60000' \
--header 'validate-appkey: xxxxxxxxxx' \
--header 'validate-timestamp: xxxxxxxxxx' \
--header 'validate-signature: xxxxxxxxxx'
```

### 说明

此接口用于查询成交记录。支持按交易对、业务类型、订单方向、订单类型、时间范围进行筛选，并支持分页查询。
