### 描述
**接口地址：**
`GET /v4/order/finance/loan/order/history/adjust-pledge`

---

### 查询参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 可选值 |
|------|------|------|--------|------|--------|
| page | integer | 否 | 1 | 页码 | — |
| limit | integer | 否 | 10 | 每页数量 | — |
| orderId | string | 否 | — | 订单ID | — |
| loanCurrencyIdList | array | 否 | — | 借款币种ID列表 | — |
| pledgeCurrencyIdList | array | 否 | — | 质押币种ID列表 | — |
| adjustPledgeType | string | 否 | — | 调整方向 | ADD（增加质押）, REMOVE（减少质押） |
| adjustPledgeRateType | string | 否 | — | 调整触发类型 | MANUAL（手动调整）, AUTO（自动调整） |
| productTypes | array | 否 | — | 产品类型列表 | SEVEN_DAY_APR, THIRTY_DAY_APR, NINETY_DAY_APR, DEMAND, VIP |
| startTime | integer | 否 | — | 开始时间戳 (毫秒) | — |
| endTime | integer | 否 | — | 结束时间戳 (毫秒) | — |

---
