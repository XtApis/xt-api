### Endpoint

`GET /future/copytrade/user/v1/copy-trade/follower-position`

---

### Parameters

| Name            | Type   | Mandatory | Default | Description       | Ranges |
| --------------- | ------ | --------- | ------- | ----------------- | ------ |
| leaderAccountId | number | Yes       | -       | Leader account ID | -      |

---

### Limit Flow Rules

2/s/apikey

---
