**类型:** GET

**描述:** `/future/market/v1/public/leverage/bracket/detail`

#### 注意：

此方法不需要签名。

### 限流规则

无

### 参数

| 名称   | 类型   | 必填 | 默认值 | 描述  | 范围 |
| ------ | ------ | --------- | ------- | ------------ | ------ |
| symbol | string | true      | N/A     | 交易对 |        |

### 请求示例

```bash title="Request"
{
  curl -G "https://fapi.xt.com/future/market/v1/public/leverage/bracket/detail" \
      -d "symbol=eth_usdt"
}
```

### 错误码

| 错误码 | 描述 |
|-------------|----------------|
| invalid_symbol | 交易对不存在 |
