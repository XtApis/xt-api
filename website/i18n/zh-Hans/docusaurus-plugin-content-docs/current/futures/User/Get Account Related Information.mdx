### 接口描述

**请求类型:** GET

**接口地址:** `/future/user/v1/account/info`

---

### 限流规则

- 200次/秒/apikey

---

### 请求参数

_无_

---

### 响应示例

```json
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "accountId": 0,           // 账户ID
    "allowOpenPosition": false, // 是否允许开仓
    "allowTrade": false,        // 是否允许交易
    "allowTransfer": false,     // 是否允许划转
    "openTime": "",             // 开户时间
    "state": 0,                 // 用户状态
    "userId": 0                 // 用户ID
  },
  "returnCode": 0
}
