### 参数

| 名称 | 类型 | 是否必填 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| | string | false | | | |

**格式**

```javascript title="格式"
{
  "topic": "trade",
  "event": "trade@btc_usdt",  // 标题
  "data":  { }
}
```

**交易记录示例（实时推送）**

```json title="交易记录示例（实时推送）"
{
  "topic": "trade",
  "event": "trade@btc_usdt",
  "data": {
    "s":  "btc_usdt",             // 交易对
    "i":  6316559590087222000,    // 交易ID
    "t":  1655992403617,          // 时间
    "oi": 6616559590087222666,    // 订单ID
    "p":  "43000",                // 价格
    "q":  "0.21",                 // 数量
    "v":  "9030",                 // 计价数量
    "b":  true                    // 是否买方做市（buyerMaker）
  }
}
```
