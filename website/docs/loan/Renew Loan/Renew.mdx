### Description

**Endpoint:** `POST /v4/order/finance/loan/order/renew`

---

### Request Parameters

<CodeBlock language="json" title="Request Parameters">
  {`{
  "bizId": 12345678910,
  "positionId": 12345,                 // Original position ID
  "productType": "THIRTY_DAY_APR",     // Renewal product type
  "couponId": 789,                     // Coupon ID (optional)
  "discountConditionSatisfied": true   // Whether discount conditions are satisfied (optional)
}`}
</CodeBlock>
