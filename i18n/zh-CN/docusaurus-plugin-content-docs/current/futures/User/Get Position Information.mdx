**请求类型** GET

**接口地址:** `/future/user/v1/position/list`

---

### 请求参数

| 参数名  | 类型   | 是否必填 | 默认值 | 描述                                     |
| ------- | ------ | -------- | ------ | ---------------------------------------- |
| symbol  | string | 否       | N/A    | 交易对（不传时查询所有交易对的持仓信息） |

---

### 限流规则

200次/秒/apikey

---

### 请求示例

```bash title="Request"
curl -G "https://fapi.xt.com/future/user/v1/position/list" \
  -H "validate-appkey: $APPKEY" \  # 必须
  -H "validate-timestamp: $TIMESTAMP" \  # 必须
  -H "validate-singature: $SINGATURE" \  # 必须
  -H "Content-Type: application/x-www-form-urlencoded" \  # 必须
  -d "symbol=btc_usdt"
```

### 响应示例

```json title="Response"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": [
    {
      "autoMargin": false,        // 是否自动追加保证金
      "availableCloseSize": 0,    // 可平仓数量（张）
      "closeOrderSize": 0,        // 挂单数量（张）
      "entryPrice": 0,            // 开仓均价
      "isolatedMargin": 0,        // 逐仓保证金
      "leverage": 0,              // 杠杆倍数
      "openOrderMarginFrozen": 0, // 占用开仓保证金
      "positionSide": "",         // 持仓方向
      "positionSize": 0,          // 持仓数量（张）
      "positionType": "",         // 仓位类型
      "realizedProfit": 0,        // 已实现盈亏
      "symbol": ""                // 交易对
    }
  ],
  "returnCode": 0
}
```

### 错误码

| Error code | Desc |
|-------------|------|
| invalid_symbol | 交易对不存在 |
