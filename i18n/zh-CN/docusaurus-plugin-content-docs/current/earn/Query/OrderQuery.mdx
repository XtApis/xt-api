**类型:** GET

**描述:** `/v4/finance/order`

根据条件筛选并查询用户的理财订单详情。

### 请求参数

| 名称 | 类型 | 必填 | 描述 |
| --- | --- | --- | --- |
| productId | long | 否 | 产品ID |
| orderType | string | 否 | 订单类型：SUBSCRIBE、REDEEM、SETTLE_INTEREST |
| status | string | 否 | 订单状态：SUCCESS、PROCESSING、FAILED |
| startTime | long | 否 | 开始时间（毫秒时间戳） |
| endTime | long | 否 | 结束时间（毫秒时间戳） |
| page | integer | 否 | 页码 |
| size | integer | 否 | 每页数量 |

### 请求示例

<CodeBlock language="bash" title="Request">
{`curl -G "https://sapi.xt.com/v4/finance/order" \\
  -H "validate-appkey: $APPKEY" \\
  -H "validate-timestamp: $TIMESTAMP" \\
  -H "validate-signature: $SIGNATURE" \\
  -H "Content-Type: application/json" \\
  -d "orderType=SUBSCRIBE" \\
  -d "page=1" \\
  -d "size=10"
`}
</CodeBlock>

### 响应字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| productName | string | 产品名称 |
| currency | string | 申购币种 |
| profitType | string | 收益类型 |
| apy | array | 参考年化收益率 |
| financeOrderType | string | 订单类型：SUBSCRIBE、REDEEM、SETTLE_INTEREST |
| amount | string | 金额 |
| productPrice | string | 产品价格 |
| startTime | long | 发起时间 |
| arrivalTime | long | 到账时间 |
| financeOrderStatus | string | 订单状态：SUCCESS、PROCESSING、FAILED |
| duration | integer | 期限（天），活期为null |
| interestCurrency | string | 收益币种 |
