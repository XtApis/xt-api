### Description

**Endpoint:** `GET /v4/public/finance/loan/product/loan-currency`

---

### Query Parameters

| Name | Type | Required | Default | Description | Allowed Values |
| --- | --- | --- | --- | --- | --- |
| page | integer | false | 1 | Page number | — |
| limit | integer | false | 10 | Number of items per page | — |
| currencyId | integer | false | — | Currency ID | — |
| sortType | string | false | — | Sort type | SEVEN_DAY_APR_DESC, SEVEN_DAY_APR_ASC, THIRTY_DAY_APR_DESC, THIRTY_DAY_APR_ASC, NINETY_DAY_APR_DESC, NINETY_DAY_APR_ASC |

---
