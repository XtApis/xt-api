**类型:** GET

**接口描述:** `/v4/batch-order`

### 请求参数

| 参数名   | 类型   | 是否必填 | 默认值 | 描述             | 取值范围 |
|----------|--------|----------|--------|------------------|----------|
| orderIds | string | 是       |        | 订单 ID 列表，例如: 6216559590087220004, 6216559590087220005 |          |

### 说明

返回字段信息与 **查询单个订单接口** 相同。

### 请求示例

```
curl --location --request DELETE 'https://sapi.xt.com/v4/batch-order' \
--header 'accept: */*' \
--header 'Content-Type: application/json' \
--header 'validate-algorithms: HmacSHA256' \
--header 'validate-recvwindow: 60000' \
--header 'validate-appkey: xxxxxxxxxx' \
--header 'validate-timestamp: xxxxxxxxxx' \
--header 'validate-signature: xxxxxxxxxx' \
--data '{
  "orderIds": [123456789, 987654321]
}'
```
