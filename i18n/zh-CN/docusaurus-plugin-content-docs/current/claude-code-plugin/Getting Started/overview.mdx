# XT Exchange Plugin for Claude Code

XT.COM 正式推出 Claude Code 交易插件，通过自然对话即可获得 XT 级别的行情洞察与交易执行能力。

## 什么是 Claude Code？

Claude Code 是 Anthropic 的 AI 编程工具。Skills 为 Claude Code 扩展了特定领域的知识 —— 当一个 skill 激活时，Claude 无需您在每次对话中粘贴文档，即可理解库的架构、API 和约定。

## 这个插件是什么？

XT Exchange Plugin 将 XT.COM 交易所的功能直接集成到 Claude Code 中。您无需手动调用 API 或编写交易脚本，可以通过自然对话与 XT.COM 进行交互。

该插件采用**渐进式交互** —— 每次只提出一个问题，而不是显示完整菜单，保持交互快速且节省上下文。

## 本次上线功能

### 现货交易（`sapi.xt.com`）

行情（最新价 / 24小时统计 / 订单簿 / K线 / 交易对信息）、账户（余额 / 当前挂单 / 历史订单）、交易（限价/市价 买卖 / 撤单 / 全部撤单）、资产（账户划转 / 提币），支持 API Key/Secret 签名。

### 合约交易（`fapi.xt.com`，USDT 永续合约）

行情（最新价 / 订单簿 / 资金费率 / K线）、账户（权益与保证金 / 持仓 / 当前委托）、交易（开多/开空 / 平仓 / 撤单 / 历史委托）。

## 核心优势

- **对话即交易**：同一套插件同时覆盖现货与合约，既能查看行情也能即时下单。
- **研究到执行一站式**：从行情查询 > 账户查看 > 下单交易，全流程无需切换工具。
- **安全确认机制**：所有下单、撤单和提币操作均需明确确认后才执行，避免误操作。
- **开发者友好**：插件完全开放，安装即用，通过自然对话即可接入 XT 级别的交易能力。

## 环境要求

- Python 3.8+
- `requests` >= 2.28.0
- 已安装 Claude Code CLI
- XT.COM 账户及 API 密钥
