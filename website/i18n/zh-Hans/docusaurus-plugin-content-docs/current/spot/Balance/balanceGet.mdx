### 描述
**GET** `/v4/balance`

---

### 请求参数

| 参数名     | 类型   | 必填 | 默认值 | 描述   | 取值范围 |
|------------|--------|------|--------|--------|----------|
| currency   | string | 是   | N/A    | 币种，例如：usdt |  |

---

### 请求示例

```json title="Request"
  curl --location --request GET 'https://sapi.xt.com/v4/balance?currency=XT' \
    --header 'accept: */*' \
    --header 'Content-Type: application/json' \
    --header 'validate-algorithms: HmacSHA256' \
    --header 'validate-recvwindow: 60000' \
    --header 'validate-appkey: xxxxxxxxxx' \
    --header 'validate-timestamp: xxxxxxxxxx' \
    --header 'validate-signature: xxxxxxxxxx' \
```

### 返回示例

```json title="Response"
{
  "rc": 0,
  "mc": "string",
  "ma": [
    {}
  ],
  "result": {
    "currency": "usdt",          // 币种
    "currencyId": 0,             // 币种ID
    "frozenAmount": 0,           // 冻结数量
    "availableAmount": 0,        // 可用数量
    "totalAmount": 0,            // 总数量
    "convertBtcAmount": 0        // 折合BTC数量
  }
}
```
