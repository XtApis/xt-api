**Type:** GET

**Description:** `/future/market/v1/public/leverage/bracket/detail`

#### Noteï¼š

This method does not require a signature.

### Limit flow rules

None

### Parameters

| Name   | Type   | Mandatory | Default | Description  | Ranges |
| ------ | ------ | --------- | ------- | ------------ | ------ |
| symbol | string | true      | N/A     | Trading pair |        |

### Request Example

```bash title="Request"
{
  curl -G "https://fapi.xt.com/future/market/v1/public/leverage/bracket/detail" \
      -d "symbol=eth_usdt"
}
```
