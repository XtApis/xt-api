### 请求方式
- 类型: **GET**
- 描述: `/future/market/v1/public/cg/contracts`
- 注意: 该方法不需要签名

---

### 请求示例

```bash
curl -G "https://fapi.xt.com/future/market/v1/public/cg/contracts" \
    -H "Content-Type: application/x-www-form-urlencoded" \  # 必须
```

---

### 响应示例

```json
[{
  "id": 123,
  "ask": "1817.32",                 // 当前最低卖价
  "base_currency": "ETH",           // 基础交易对的货币代码，如 BTC
  "base_volume": "13267684284",     // 24小时交易量
  "bid": "1817.31",                 // 当前最高买价
  "contractSize": 10,               // 合约面值
  "end_timestamp": 253402099200000, // 合约到期时间
  "funding_rate": "-0.03",          // 资金费率
  "high": "1828.89",                // 24小时最高成交价
  "index_currency": "USD",          // 指数标的货币
  "index_name": "ETH-USD",          // 标的指数名称（如有）
  "index_price": "1816.61",         // 标的指数价格
  "last_price": "1817.31",          // 最新成交价
  "low": "1778.65",                 // 24小时最低成交价
  "next_funding_rate": "-0.03",     // 下次预测资金费率
  "next_funding_rate_timestamp":1698681600000,  // 下次资金费率时间
```
