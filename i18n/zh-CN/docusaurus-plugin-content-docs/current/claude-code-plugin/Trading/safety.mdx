# 安全与声明

## 如何使用

在 Claude Code 中输入 `/xt-exchange`，然后按照提示操作。

该插件采用**渐进式交互** —— 每次只提出一个问题，而不是显示完整菜单，保持交互快速且节省上下文。

## 安全机制

### 下单确认

所有下单、撤单和划转操作**均需明确确认**后才会执行。插件会显示完整的订单详情，并要求您确认后再执行。

### 提币保护

提币操作**不可逆**。执行前，插件会显示完整详情，包括：

- 币种和链
- 金额和手续费
- 目标地址

任何提币操作执行前都需要明确确认。

### 凭证安全

- API 凭证存储在本地 `~/.xt-exchange/credentials.json`
- 文件权限设置为**仅所有者可读**（`chmod 600`）
- 凭证不会传输给任何第三方 —— 仅用于直接调用 XT.COM API

## 最佳实践

1. **从只读开始** —— 初始仅启用 **读取** 权限，熟悉插件功能后再开启其他权限
2. **使用 IP 白名单** —— 在 XT.COM API 管理页面限制 API 密钥的访问 IP
3. **小额测试优先** —— 执行大额交易前先用小额测试
4. **确认前仔细核查** —— 始终仔细核查插件显示的订单详情后再确认

## 注意事项

- 通过本插件下的订单由 AI 基于自然语言解析用户指令后发起，AI 的理解可能与您的意图存在偏差，请在确认前仔细核查订单详情。
- 加密货币交易涉及市场风险，价格可能剧烈波动。本插件不提供投资建议，请始终自行做好研究（DYOR）。
- 某些操作（如提币）一旦执行即不可撤销，请格外谨慎。
- 使用本插件即表示您理解并接受上述风险，请详细参阅 [XT.COM 使用条款](https://xtsupport.zendesk.com/hc/en-us/articles/900001631023-User-Agreement)。
