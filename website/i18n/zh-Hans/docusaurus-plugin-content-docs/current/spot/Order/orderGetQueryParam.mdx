### 描述
**请求类型:** GET

**请求路径:** `/v4/order`

---

### 请求参数

| 名称          | 类型   | 必填 | 默认值 | 描述       |
|---------------|--------|------|--------|------------|
| orderId       | number | 否   | N/A    | 订单 ID    |
| clientOrderId | string | 否   | N/A    | 客户订单 ID |

---

### 限流规则
- 50 次/秒/每个 API Key
