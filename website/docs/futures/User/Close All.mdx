**Type:** POST

**Description:** `/future/user/v1/position/close-all`

#### Limit Flow Rules

200/s/apikey

### Request Parameters

None

### Request Example

```bash title="Request"
curl -X POST "https://fapi.xt.com/future/user/v1/position/close-all" \ 
  -H "validate-appkey: $APPKEY" \  # Required
  -H "validate-timestamp: $TIMESTAMP" \  # Required
  -H "validate-singature: $SINGATURE" \  # Required
  -H "Content-Type: application/x-www-form-urlencoded"  # Required
```

### Response Example

```json title="Response"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": true,
  "returnCode": 0
}
```
