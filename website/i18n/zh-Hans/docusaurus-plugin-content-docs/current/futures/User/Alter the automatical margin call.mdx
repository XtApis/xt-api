### 接口描述

**POST** `/future/user/v1/position/auto-margin`

`Content-Type = application/x-www-form-urlencoded && application/json`

---

### 限流规则
- 200次/秒/apikey

---

### 请求参数

| 参数名       | 类型    | 是否必填 | 默认值 | 描述               | 取值范围    |
|--------------|---------|----------|--------|--------------------|-------------|
| symbol       | string  | 是       | N/A    | 交易对             |             |
| positionSide | string  | 是       | N/A    | 持仓方向           | LONG; SHORT |
| autoMargin   | boolean | 是       | N/A    | 是否自动追加保证金 | true; false |

---

### 请求示例

```bash title="Request"
curl -X POST "https://fapi.xt.com/future/user/v1/position/auto-margin" \
  -H "validate-appkey: $APPKEY" \  # 必须
  -H "validate-timestamp: $TIMESTAMP" \  # 必须
  -H "validate-singature: $SINGATURE" \  # 必须
  -H "Content-Type: application/x-www-form-urlencoded" \  # 必须
  -d "symbol=btc_usdt" \  # 必须
  -d "positionSide=SHORT" \  # 必须
  -d "autoMargin=true" \  # 必须
```

### 响应示例

```json
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {},
  "returnCode": 0
}
```

### 错误码

| Error code | Desc |
|-------------|------|
| invalid_symbol | 交易对不存在/不可为空 |
| invalid_position_side | 仓位方向不可为空 |
| invalid_auto_margin | 自动追加保证金配置不可为空 |
