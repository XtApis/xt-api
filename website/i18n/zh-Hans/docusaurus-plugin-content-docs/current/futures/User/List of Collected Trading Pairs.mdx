**请求类型** GET

**接口地址：** `/future/user/v1/user/collection/list`

---

### 限流规则

200次/秒/apikey

---

### 请求示例
```bash
  curl -G "https://fapi.xt.com/future/user/collection/list" \ 
  -H "validate-appkey: $APPKEY" \  # 必须
  -H "validate-timestamp: $TIMESTAMP" \  # 必须
  -H "validate-singature: $SINGATURE" \  # 必须
  -H "Content-Type: application/x-www-form-urlencoded" \  # 必须
```

---

### 响应示例

```json title="Response"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": [],
  "returnCode": 0
}
```
