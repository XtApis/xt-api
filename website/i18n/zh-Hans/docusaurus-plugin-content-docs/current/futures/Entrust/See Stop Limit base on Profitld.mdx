**请求类型:** GET
**描述:** `/future/trade/v1/entrust/profit-detail`

---

### 限流规则
200/s/apikey

---

### 参数说明
| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
|------|------|------|--------|------|------|
| profitId | integer | TRUE | N/A | 止盈止损订单ID | — |

---

### 请求示例
```bash
curl -G "https://fapi.xt.com/future/trade/v1/entrust/profit-detail" \
    -H "validate-appkey: $APPKEY"              # 必须
    -H "validate-timestamp: $TIMESTAMP"        # 必须
    -H "validate-signature: $SIGNATURE"        # 必须
    -H "Content-Type: application/x-www-form-urlencoded"  # 必须
    -d "profitId=1234567890"                   # 必须
```

---

### 响应示例
```json
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "createdTime": 0,                // 时间
    "entryPrice": 0,                 // 开仓平均价格
    "executedQty": 0,                // 实际成交
    "isolatedMargin": 0,             // 逐仓保证金
    "origQty": 0,                    // 数量（张）
    "positionSide": "",              // 持仓方向
    "positionSize": 0,               // 持仓数量（张）
    "profitId": 0,                   // 订单ID
    "state": "",                     // 订单状态：NOT_TRIGGERED（未触发）；TRIGGERING（触发中）；TRIGGERED（已触发）；USER_REVOCATION（用户撤销）；PLATFORM_REVOCATION（平台撤销）；EXPIRED（已过期）
    "symbol": "",                    // 交易对
    "triggerProfitPrice": 0,         // 止盈价格
    "triggerStopPrice": 0            // 止损价格
  },
  "returnCode": 0
}
```

---

### 错误码
| 错误码 | 描述 |
|---------|------|
| invalid_profit_id | Profit ID 不存在 |
