### 概述

本文档描述理财系统的 API 接口，支持理财产品申购、赎回、持仓查询及自动化开关控制功能。

**支持的产品类型：**

| 产品类型 | 支持申购 | 支持赎回 | 备注 |
| --- | --- | --- | --- |
| 活期 | 是 | 是 | 随时申购/赎回，支持自动申购/一键理财开关 |
| 定期 | 是 | 否 | 随时申购，不可赎回，支持自动续投开关 |
| 抢购 | 是 | 否 | 仅申购期间开放申购，不可赎回 |
| 链上赚币 | 是 | 是 | 随时申购/赎回，赎回有赎回期 |
| 链上精选XRWU | 是 | 是 | 含标准赎回（T+3到账）与快速赎回（即刻到账） |
| 财富管理 | 是 | 是 | 随时申购/赎回，赎回需等待下一个赎回日到账 |
| 鲨鱼鳍 | 是 | 否 | 仅申购期间可申购，不可赎回，支持自动续投开关 |
| 趋势智盈 | 是 | 否 | 仅申购期间可申购，不可赎回，支持自动续投开关 |

**基本信息：**

- **Base URL:** `https://sapi.xt.com`
- **认证方式:** API Key + 签名
- **Content Type:** `application/json`

### 认证

所有 API 请求必须在请求头中包含认证信息。API 需要启用**理财权限**。
