**请求类型** POST

**接口描述:** `/future/user/v1/position/change-type`

### 限流规则

200次/秒/apikey

### 请求参数

| 参数名       | 类型   | 是否必填 | 默认值 | 描述     | 取值范围            |
| ------------ | ------ | -------- | ------ | -------- | ------------------- |
| symbol       | string | 是       | N/A    | 交易对   |                     |
| positionSide | string | 是       | N/A    | 持仓方向 | LONG; SHORT         |
| positionType | string | 是       | N/A    | 仓位模式 | CROSSED; ISOLATED   |

### 响应示例

```json title="Response"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {},
  "returnCode": 0
}
