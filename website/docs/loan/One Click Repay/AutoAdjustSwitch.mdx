### Description

**Endpoint:** `GET /v4/order/finance/loan/order/on-click`

---

### Query Parameters

- `productTypes`: List of product types

**Product Type Description:**

- `NINETY_DAY_APR` - Represents current (demand) product
- `VIP` - Represents VIP product
- Fixed-term products can omit this parameter

---
