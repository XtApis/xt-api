### Request format

```
agg_ticker@{symbol}
```

- Example: `agg_ticker@btc_usdt`
- Push rate: `1000ms`

---

### Response example

```json
{
  "topic": "agg_ticker",
  "event": "agg_ticker@btc_usdt",
  "data": {
    "s": "btc_index", // symbol

    "c": "50000", // close
    "h": "0.1", // high
    "l": "0.1", // low
    "a": "0.1", // volume

    "m": "0.21", // mark price
    "bp": "0.21", // best bid

    "t": 123124124 // timestamp
  }
}
```
