### 接口描述
**GET** `/v4/public/currencies`

---

### 请求参数

| 参数名 | 类型   | 是否必填 | 默认值 | 描述             | 取值范围 |
|--------|--------|----------|--------|------------------|----------|
| —      | string | 否       | N/A    | 无需请求参数     |          |

---

### 返回示例

```json
{
  "rc": 0,
  "mc": "string",
  "ma": [{}],
  "result": [
    {
      "id": 11,                 // 币种 ID
      "currency": "usdt",       // 币种简称
      "fullName": "usdt",       // 币种全称
      "logo": null,             // 币种 Logo
      "cmcLink": null,          // CMC 链接
      "weight": 100,            // 币种权重
      "maxPrecision": 6,        // 最大精度
      "depositStatus": 1,       // 充值状态 (0 关闭，1 开启)
      "withdrawStatus": 1,      // 提现状态 (0 关闭，1 开启)
      "convertEnabled": 1,      // 小额资产兑换 (0 关闭，1 开启)
      "transferEnabled": 1      // 转账开关 (0 关闭，1 开启)
    }
  ]
}
