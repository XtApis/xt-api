**类型:** GET

**描述:** `/v4/user/account/api-key/broker-sub`

查询经纪商名下子账户的 API Key。请求必须使用**经纪商母账户**的 API Key 签名。返回的 `accessKey` 已脱敏，且不返回 `secretKey`。

### 参数

| 名称         | 类型 | 是否必填 | 默认值 | 描述                                                 | 取值范围 |
| ------------ | ---- | -------- | ------ | ---------------------------------------------------- | -------- |
| subAccountId | long | 否       | -      | 按子账户过滤；不传则列出该经纪商所有子账户的 API Key | -        |

---

### 响应参数

| 名称         | 类型     | 描述                  |
| ------------ | -------- | --------------------- |
| subAccountId | long     | 子账户 id             |
| accessKey    | string   | 脱敏后的 access key   |
| permissions  | string[] | 权限码                |
| bindIps      | string[] | 已绑定的 IP           |
| status       | string   | `active` / `disabled` |
| createTime   | long     | 创建时间（毫秒）      |

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": [
    {
      "subAccountId": 10049631158745,
      "accessKey": "69af****0840",
      "permissions": ["QUERY"],
      "bindIps": [],
      "status": "active",
      "createTime": 1784088472976
    }
  ],
  "success": true
}
```
