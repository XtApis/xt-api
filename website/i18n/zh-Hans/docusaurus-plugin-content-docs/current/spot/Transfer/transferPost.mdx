**类型：** POST

**接口描述：** `/v4/balance/transfer`

### 请求参数

| 名称 | 类型 | 是否必填 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| bizId | string | 是 | N/A | 用于幂等处理的唯一ID | 最大长度：128 |
| from | enum | 是 | N/A | 资金转出账户 | [bizType 枚举](#bizType) |
| to | enum | 是 | N/A | 资金转入账户 | [bizType 枚举](#bizType) |
| currency | string | 是 | N/A | 币种名称，必须小写（例如 `usdt`、`btc`） |  |
| symbol | string | 否 | N/A | 划转交易对（必须小写）。如果转入或转出账户中有一个是**杠杆账户**，则必填 |  |
| amount | bigDecimal | 是 | N/A | 划转金额 |  |

### 注意事项

此接口用于同一用户下**不同业务系统**之间的资金划转。

- `bizId` 确保幂等处理，建议保存用于对账。
- 如涉及杠杆账户，必须提供 `symbol` 参数。


####  请求示例
```
curl --location --request POST 'https://sapi.xt.com/v4/balance/transfer' \
--header 'accept: */*' \
--header 'Content-Type: application/json' \
--header 'validate-algorithms: HmacSHA256' \
--header 'validate-recvwindow: 60000' \
--header 'validate-appkey: xxxxxxxxxx' \
--header 'validate-timestamp: xxxxxxxxxx' \
--header 'validate-signature: xxxxxxxxxx' \
--data '{
    "bizId": "xxxx_xxxxxxxx",
    "from": "SPOT",
    "to": "FUTURES_U",
    "currency": "USDT",
    "amount": "1"
}'
```

### 响应示例

```json title="响应"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": 123456 // 返回的划转唯一ID，建议保存用于对账
}
```
