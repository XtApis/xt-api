### 订阅 / 取消订阅

```json title="订阅 / 取消订阅"
{
  "method": "SUBSCRIBE/UNSUBSCRIBE",
  "params": ["balance@{ListenKey}"],
  "id": "{id}"
}
```

### 响应

```json title="响应示例"
{
  "topic": "balance",
  "event": "balance@123456",
  "data": {
    "coin":                  "usdt",
    "underlyingType":        1,      // 1: 币本位，2: USDT本位
    "walletBalance":         "123", // 余额
    "openOrderMarginFrozen": "123", // 冻结订单
    "isolatedMargin":        "213", // 逐仓保证金
    "crossedMargin":         "0"    // 全仓保证金
  }
}
```
