**类型:** POST

**描述:** `/future/trade/v1/order/cancel-all`

### 限流规则

200/s/apikey

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | string | true | N/A | 交易对（传入 `""` 则撤销所有交易对的订单） |  |

### 请求示例

<CodeBlock language="bash" title="Request">
{`curl -X POST "https://fapi.xt.com/future/trade/v1/order/cancel-all" \
  -H "validate-appkey: $APPKEY" \  # 必须
  -H "validate-timestamp: $TIMESTAMP" \  # 必须
  -H "validate-singature: $SINGATURE" \  # 必须
  -H "Content-Type: application/x-www-form-urlencoded" \  # 必须
  -d "symbol=eth_usdt"  # 必须
`}
</CodeBlock>

### 响应示例

<CodeBlock language="json" title="Response">
  {`{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": true,
  "returnCode": 0
}`}
</CodeBlock>

### 错误码

| 错误码 | 描述 |
| --- | --- |
| invalid_symbol | 交易对不存在 |
| fail | 内部异常 |
