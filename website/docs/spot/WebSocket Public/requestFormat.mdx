### Subscribe

```json title="Subscribe"
{
  "method": "subscribe",
  "params": [
    "{topic}@{arg},{arg}",
    "{topic}@{arg}"
  ],
  "id": "{id}" // call back ID
}

Unsubscribe
{
  "method": "unsubscribe",
  "params": [
    "\{topic\}@\{arg\},\{arg\}"
  ],
  "id": "\{id\}" // call back ID
}
Response Example
{
  "code": 0,
  "msg": "success"
}
```
