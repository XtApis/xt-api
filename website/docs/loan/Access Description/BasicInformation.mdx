### Overview

This document describes the API interfaces of the loan system, which support core functionalities such as borrowing, repayment, pledge adjustment, and renewal.

**Basic Information:**

- **Base URL:** `https://sapi.xt-dev.com`
- **Authentication Method:** Gateway Header
- **Content Type:** `application/json`
