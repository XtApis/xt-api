**Type:** GET

**Description:** `/v4/referal/invite/commissions`

### Parameters

| Name | Type   | Mandatory | Default | Description | Ranges |
| ---- | ------ | --------- | ------- | ----------- | ------ |
| startTime  | string | False      |     None    | Start time (milliseconds timestamp)    |   e.g.: 1704067200000   |
| endTime  | string | False      |     None    | End time (milliseconds timestamp)     | e.g.: 1706659200000 |
| uid  | number | true      |    None   | User ID     | Unique User Identifier   |
