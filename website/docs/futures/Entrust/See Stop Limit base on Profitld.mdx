### Parameters

| Name     | Type    | Mandatory | Default | Description              | Ranges |
| -------- | ------- | --------- | ------- | ------------------------ | ------ |
| profitId | integer | TRUE      | N/A     | Profit and Stop order ID | â€”      |

---

### Request Example

```bash
curl -G "https://fapi.xt.com/future/trade/v1/entrust/profit-detail" \
    -H "validate-appkey: $APPKEY"              # Required
    -H "validate-timestamp: $TIMESTAMP"        # Required
    -H "validate-signature: $SIGNATURE"        # Required
    -H "Content-Type: application/x-www-form-urlencoded"  # Required
    -d "profitId=1234567890"                   # Required
```

---

### Response Example

```json
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "createdTime": 0, // Time
    "entryPrice": 0, // Average entry price
    "executedQty": 0, // Executed quantity
    "isolatedMargin": 0, // Isolated margin
    "origQty": 0, // Quantity (contracts)
    "positionSide": "", // Position side
    "positionSize": 0, // Position size (contracts)
    "profitId": 0, // Order ID

    "symbol": "", // Trading pair
    "triggerProfitPrice": 0, // Take-profit price
    "triggerStopPrice": 0 // Stop-loss price
  },
  "returnCode": 0

```
