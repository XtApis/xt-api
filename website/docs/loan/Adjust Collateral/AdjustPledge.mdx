**Type:** POST

**Description:** `/v4/order/finance/loan/order/v1/borrow`

### 注意事项

**仅支持普通借款，不包含 VIP 借款和新手借款**

### 参数说明

### 请求示例:

```json
{
  "loanProductId": 123, // 借款产品ID
  "pledgeProductId": 456, // 质押产品ID
  "productType": "SEVEN_DAY_APR", // 产品类型: SEVEN_DAY_APR, THIRTY_DAY_APR, DEMAND
  "loanAmount": "30000", // 借款金额
  "pledgeAmount": "1.5", // 质押金额
  "couponId": 789, // 优惠券ID (可选)
  "isNovice": false // 是否新手订单
}
```

### 响应示例

```json
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": "542798144228902336"
}
```
