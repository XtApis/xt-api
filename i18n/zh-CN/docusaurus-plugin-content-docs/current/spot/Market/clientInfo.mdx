**类型:** GET **描述:** `/v4/public/client`

### 请求示例

```bash title="Request"
  curl --location --request GET 'https://sapi.xt.com/v4/public/client' \
    --header 'accept: */*' \
    --header 'Content-Type: application/json' \
```

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": {
    "ip": "192.168.1.1"
  }
}
```
