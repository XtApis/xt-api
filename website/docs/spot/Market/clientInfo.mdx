**Type:** GET **Description:** `/v4/public/client`

### Request Example

```bash title="Request"
  curl --location --request GET 'https://sapi.xt.com/v4/public/client' \
    --header 'accept: */*' \
    --header 'Content-Type: application/json' \
```

### Response Example

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": {
    "ip": "192.168.1.1"
  }
}
```
