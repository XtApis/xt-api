**请求类型:** POST
**描述:** `/future/trade/v1/entrust/cancel-track`

---

### 限流规则

200/s/apikey

---

### 参数说明
| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
|------|------|------|--------|------|------|
| trackId | integer | TRUE | N/A | 跟踪单ID | — |

---

### 请求示例
```bash
curl -X POST "https://fapi.xt.com/future/trade/v1/entrust/cancel-track" \
    -H "validate-appkey: $APPKEY"              # 必须
    -H "validate-timestamp: $TIMESTAMP"        # 必须
    -H "validate-signature: $SIGNATURE"        # 必须
    -H "Content-Type: application/x-www-form-urlencoded"  # 必须
    -d "trackId=1234567890"                    # 必须
```

---

### 响应示例
```json
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {},
  "returnCode": 0
}
```

---

### 错误码
| 错误码 | 描述 |
|---------|------|
| invalid_track_entrust | 跟踪单不存在 |
