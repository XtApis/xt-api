**类型:** GET

**描述:** `/future/market/v1/public/q/mark-price`

#### 注意：

此方法不需要签名。

### 限流规则

无

### 参数

无

### 请求示例

```bash title="Request"
{
  curl -G "https://fapi.xt.com/future/market/v1/public/q/mark-price" 
}
```
