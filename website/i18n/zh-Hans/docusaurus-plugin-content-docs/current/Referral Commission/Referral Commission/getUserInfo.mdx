**类型:** GET

**描述:** `/v4/referal/invite/single/user/info`

### 参数

| 名称 | 类型   | 是否必需 | 默认值 | 描述 |
| ---- | ------ | --------- | ------- | ----------- |
| uid  | number | true      |         | 用户ID     |

---
