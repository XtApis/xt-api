**类型:** POST

**描述:** `/future/trade/v2/order/atomic-create-batch`

> Content-Type = `application/x-www-form-urlencoded && application/json`

### 限流规则

200/s/apikey

### 参数

请求体（Body）

请求体为 JSON 数组，每个元素代表一笔独立订单。

### 请求示例

<CodeBlock language="bash" title="Request">
{`curl -X POST "https://fapi.xt.com/future/trade/v2/order/atomic-create-batch" \\
  -H "validate-appkey: $APPKEY" \\  # 必须
  -H "validate-timestamp: $TIMESTAMP" \\  # 必须
  -H "validate-signature: $SINGATURE" \\  # 必须
  -H "Content-Type: application/json" \\  # 必须
  -d '[
        {
          "symbol": "btc_usdt",
          "origQty": 10,
          "orderType": "MARKET",
          "orderSide": "BUY",
          "positionSide": "LONG"
        },
        {
          "symbol": "eth_usdt",
          "origQty": 5,
          "orderType": "MARKET",
          "orderSide": "BUY",
          "positionSide": "LONG"
        }
      ]' \\  # 必须
`}
</CodeBlock>

### 响应示例

<CodeBlock language="json" title="Response">
{`{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": "",
  "returnCode": 0
}`}
</CodeBlock>
