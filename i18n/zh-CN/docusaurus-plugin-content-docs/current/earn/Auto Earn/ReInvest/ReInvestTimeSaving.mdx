**类型:** POST

**描述:** `/v4/finance/re-invest-timesaving`

开启或关闭定期产品的自动复投。

### 请求参数

| 名称 | 类型 | 必填 | 描述 |
| --- | --- | --- | --- |
| positionId | string | 是 | 持仓ID |
| enabled | boolean | 是 | 开启或关闭复投 |

### 请求示例

<CodeBlock language="bash" title="Request">
{`curl -X POST "https://sapi.xt.com/v4/finance/re-invest-timesaving" \\
  -H "validate-appkey: $APPKEY" \\
  -H "validate-timestamp: $TIMESTAMP" \\
  -H "validate-signature: $SIGNATURE" \\
  -H "Content-Type: application/json" \\
  -d '{
    "positionId": "123456789",
    "enabled": true
  }'
`}
</CodeBlock>

### 响应示例

<CodeBlock language="json" title="Response">
{`{
  "rc": 0,
  "mc": "SUCCESS",
  "result": "success"
}`}
</CodeBlock>

### 响应字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| result | string | 操作结果 |
