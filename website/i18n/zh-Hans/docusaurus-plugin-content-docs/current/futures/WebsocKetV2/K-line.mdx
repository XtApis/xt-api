### 请求格式

```
kline@{symbol},{interval}
```

- **时间间隔选项**：`1m`、`3m`、`5m`、`15m`、`30m`、`1h`、`2h`、`4h`、`6h`、`8h`、`12h`、`1d`、`3d`、`1w`、`1M`
- **示例**：`kline@btc_usdt,5m`
- **推送频率**：1000ms

---

### 响应示例

```json title="响应"
{
  "topic": "kline",
  "event": "kline@btc_usdt,5m",
  "data": {
    "s": "btc_index", // 交易对
    "o": "49000", // 开盘价
    "c": "50000", // 收盘价
    "h": "0.1", // 最高价
    "l": "0.1", // 最低价
    "a": "0.1", // 成交量
    "v": "0.1", // 成交额
    "ch": "0.21", // 涨跌幅
    "t": 123124124 // 时间戳
  }
}
```
