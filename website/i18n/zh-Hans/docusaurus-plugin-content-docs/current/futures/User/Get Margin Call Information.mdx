**类型** GET

**接口说明：** `/future/user/v1/position/break-list`

### 限流规则

200次/秒/apikey

### 响应示例

```json title="响应"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": [
    {
      "breakPrice": 0, //强平价格。0表示无强平风险
      "calMarkPrice": 0, //标记价格
      "contractType": "", //合约类型：PERPETUAL；PREDICT
      "entryPrice": 0, //平均开仓价格
      "isolatedMargin": 0, //逐仓保证金
      "leverage": 0, //杠杆
      "positionSide": "", //持仓方向：LONG；SHORT
      "positionSize": 0, //持仓数量（张）
      "positionType": "", //持仓类型：CROSSED；ISOLATED
      "symbol": "" //交易对
    }
  ],
  "returnCode": 0
}
```
