**类型:** GET

**描述:** `/future/market/v1/public/q/kline`

#### 注意：

此方法不需要签名。

### 限流规则

无

### 参数

| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | string | true | N/A | 交易对 |  |
| interval | string | true | N/A | 时间间隔 | 1m;5m;15m;30m;1h;4h;1d;1w |
| startTime | integer | false | N/A | 开始时间 |  |
| endTime | integer | false | N/A | 结束时间 |  |
| limit | integer | false | N/A | 限制数量 |  |

### 请求示例

```bash title="Request"
{
  curl -G "https://fapi.xt.com/future/market/v1/public/q/kline" \
      -d "symbol=eth_usdt" \ # 必须
      -d "interval=1m"  # 必须 
}
```

### 错误码

| 错误码 | 描述 |
|-------------|----------------|
| invalid_symbol | 交易对不存在 |
| invalid_limit | limit 最小为 1，最大为 1500，且为整数 |
| invalid_interval | 时间间隔不合法 |
| invalid_time | 起止时间有误 |
