**请求类型** POST

**接口描述：** `/future/user/v1/position/margin`

**请求参数**

| 参数名       | 类型   | 是否必填 | 默认值 | 描述 |
| ------------ | ------ | -------- | ------ | ---- |
| symbol       | string | 是       | N/A    | 交易对 |
| margin       | number | 否       | N/A    | 数量 |
| positionSide | string | 否       | N/A    | 持仓方向 (LONG, SHORT) |
| type         | string | 否       | N/A    | 调整方向（ADD: 增加逐仓保证金, SUB: 减少逐仓保证金） |

### 限流规则

200次/秒/apikey

### 响应示例

```json title="Response"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {},
  "returnCode": 0
}
