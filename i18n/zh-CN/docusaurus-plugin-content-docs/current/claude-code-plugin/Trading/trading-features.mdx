# 交易功能

在 Claude Code 中输入 `/xt-exchange`，然后描述您想做的操作即可。

## 现货交易

该插件通过 XT.COM 现货 API（`sapi.xt.com`）进行现货交易操作。

### 行情数据

通过自然对话查询实时行情信息：

- **最新价** — 获取任意交易对的最新成交价
- **24小时统计** — 成交量、最高价、最低价、24小时涨跌幅
- **订单簿** — 当前买卖盘口深度
- **K线** — 可配置时间间隔的K线蜡烛图数据
- **交易对信息** — 交易对详情（最小下单量、价格精度等）

### 账户

查看您的账户状态：

- **余额** — 查看所有币种或指定币种的可用和冻结余额
- **当前挂单** — 列出所有当前活跃的订单
- **历史订单** — 查询历史成交或已撤销的订单

### 交易

通过确认式交互执行现货交易：

- **限价买入/卖出** — 以指定价格挂限价单
- **市价买入/卖出** — 以市价立即成交
- **撤单** — 按订单 ID 撤销指定挂单
- **全部撤单** — 撤销指定交易对的所有挂单

### 资产

跨账户管理您的资产：

- **账户划转** — 在账户类型之间划转资金（SPOT / LEVER / FUTURES_U / FINANCE）
- **提币** — 提币到外部地址（需明确确认）

### 使用示例

- "BTC/USDT 的当前价格是多少？"
- "查看我的 USDT 余额"
- "以市价买入 0.01 个 BTC"
- "撤销我 ETH/USDT 的所有挂单"

## 合约交易

该插件通过 XT.COM 合约 API（`fapi.xt.com`）进行 USDT 永续合约交易。

### 行情数据

查询合约行情信息：

- **最新价** — 获取任意合约的最新价格
- **订单簿** — 合约当前买卖盘口深度
- **资金费率** — 当前及历史资金费率
- **K线** — 可配置时间间隔的合约K线数据

### 账户

查看您的合约账户状态：

- **权益与保证金** — 查看总权益、可用保证金和保证金使用率
- **持仓** — 列出所有当前持仓及未实现盈亏
- **当前委托** — 列出所有活跃的合约订单

### 交易

通过确认式交互执行合约交易：

- **开多** — 开多头仓位（限价或市价）
- **开空** — 开空头仓位（限价或市价）
- **平仓** — 平仓（部分或全部）
- **撤单** — 撤销指定的合约挂单
- **历史委托** — 查询历史合约订单记录

### 重要说明

:::info 合约面值
合约数量以**张**（整数）为单位计算。不同合约品种的合约面值不同：

| 品种 | 合约面值 |
|------|---------|
| BTC | 每张 0.001 BTC |
| ETH | 每张 0.01 ETH |

其他品种的合约规格请查看具体合约说明。
:::

:::info 划转账户类型
向合约账户划转资金时，使用以下账户类型标识：

| 类型 | 说明 |
|------|------|
| `SPOT` | 现货账户 |
| `LEVER` | 杠杆账户 |
| `FUTURES_U` | USDT 永续合约账户 |
| `FINANCE` | 理财账户 |
:::
