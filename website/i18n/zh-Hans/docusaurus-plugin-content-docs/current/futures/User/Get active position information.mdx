**请求类型** GET

**接口地址:** `/future/user/v1/position`

---

### 请求参数

| 参数名   | 类型   | 是否必填 | 默认值 | 描述 |
| -------- | ------ | -------- | ------ | ---- |
| symbol   | string | 否       | N/A    | 交易对（不传时查询所有交易对的持仓信息） |

---

### 限流规则

200次/秒/apikey

---

    "code": "",

  "msgInfo": "",
