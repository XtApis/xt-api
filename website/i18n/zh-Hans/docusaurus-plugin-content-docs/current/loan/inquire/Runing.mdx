### 描述
**接口地址：**
`GET /v4/order/finance/loan/order/running`

---

### 查询参数

- `page`: 页码 (默认 1)
- `size`: 每页数量 (默认 10)
- `productTypes`: 产品类型列表
- `nextId`: 下一页 ID (可选)
