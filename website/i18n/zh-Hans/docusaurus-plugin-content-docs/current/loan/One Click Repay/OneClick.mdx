### 描述
**接口地址：**
`POST /v4/order/finance/loan/order/repay/on-click`

---

### 请求参数

<CodeBlock language="json" title="请求参数">
{`{
  "repayItems": [                      // 还款项目列表
    {
      "loanCurrency": "usdt",
      "productId": 123
    }
  ],
  "productTypes": ["SEVEN_DAY_APR"],   // 产品类型列表
  "bizId": "batch_repay_123456789"     // 批量还款业务ID
}`}
</CodeBlock>

---

### 响应示例

<CodeBlock language="json" title="响应示例">
{`{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": "order-id-123"
}`}
</CodeBlock>
