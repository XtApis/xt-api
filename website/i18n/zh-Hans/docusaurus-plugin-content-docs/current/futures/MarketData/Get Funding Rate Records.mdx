**类型:** GET

**描述:** `/future/market/v1/public/q/funding-rate-record`

#### 注意：

此方法不需要签名。

### 限流规则

无

### 参数

| 名称 | 类型 | 是否必需 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| symbol | string | 否 | N/A | 交易对 |  |
| direction | string | 否 | NEXT | 方向（PREV：上一页；NEXT：下一页） | PREV;NEXT |
| id | integer | 否 | N/A | id |  |
| limit | integer | 否 | 10 | 限制条数 |  |

### 请求示例

```bash title="Request"
{
  curl -G "https://fapi.xt.com/future/market/v1/public/q/funding-rate-record" \
      -d "symbol=eth_usdt" \
      -d "limit=5"
}
```

### 错误码

| 错误码 | 描述 |
|-------------|----------------|
| invalid_symbol | 交易对不存在 |
| invalid_direction | 方向填写有误 |
