### Description

**Endpoint:** `GET /v4/balance/finance/loan/asset/loan-asset-item`

---

### Query Parameters

| Name | Type | Required | Default | Description | Allowed Values |
| --- | --- | --- | --- | --- | --- |
| page | integer | true | — | Page number | — |
| limit | integer | false | 10 | Number of items per page | — |
| loanType | string | true | — | Loan type | LOAN (Debt), PLEDGE (Collateral) |

---
