**类型：** POST

**描述：** `/v4/lever/order`

### 参数

| 名称 | 类型 | 必需 | 默认值 | 描述 | 范围 |
| --- | --- | --- | --- | --- | --- |
| MarketName | String | 是 | - | 交易对，例如 `BTC_USDT` | - |
| CoinName | String | 是 | - | 币种名称，小写。例如 `btc` | - |
| Amount | Number | 是 | - | 金额，例如 `8000` | - |
| Type | String | 是 | - | [订单类型](../Access%20Description/publicModule#order-type)，支持：`LOAN` / `REPAY` | - |
| Media | String | 否 | - | 媒体来源，例如 `btok` | - |
| MediaChannel | String | 否 | - | 媒体渠道 | - |

### 注意事项

#### **限流规则**

30/s/apikey

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": {}
}
```
