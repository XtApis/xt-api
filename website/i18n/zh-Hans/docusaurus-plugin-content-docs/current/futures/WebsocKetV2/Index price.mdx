## Index Price ✅

### Request format

```
index_price@{symbol}
```

- Example: `index_price@btc_usdt`
- Push rate: `1000ms`

---

### Request example

**Subscribe** — After establishing the WebSocket connection, send the following JSON string.

```json title="SUBSCRIBE"
{
  "method": "SUBSCRIBE",
  "params": [
    "index_price@btc_usdt"
  ],
  "id": "test"     // this param could call whatever you want
}
```

---

### Response example

```json
{
  "topic": "index_price",
  "event": "index_price@btc_usdt",
  "data": {
    "s": "btc_usdt", // symbol
    "p": "50000",    // price
    "t": 123124124   // timestamp
  }
}
```
