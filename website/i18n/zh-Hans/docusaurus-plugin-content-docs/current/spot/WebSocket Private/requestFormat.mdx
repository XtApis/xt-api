**类型：** Websocket 信息 **描述：** WebSocket 请求消息的标准格式

### 参数

| 名称 | 类型 | 是否必须 | 默认值 | 描述     |
| ---- | ---- | -------- | ------ | -------- |
| –    | –    | false    | N/A    | 无需参数 |

### 说明

- 请求消息用于**订阅**或**取消订阅** WebSocket 主题。
- 参数格式：\{topic\}@\{arg\},\{arg\},…
- `listenKey` 必须通过 **/v4/ws-token** 接口获取。
- `id` 是用于匹配响应的自定义请求标识符。

### 订阅示例

```javascript title="订阅"
{
"method": "subscribe",
"params": [
  "\{topic\}@\{arg\},\{arg\}",   // 事件订阅
  "\{topic\}@\{arg\}"
],
"listenKey": "512312356123123123", // 通过 /v4/ws-token 获取的监听密钥
"id": "\{id\}"
}
```
