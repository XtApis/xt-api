**类型:** GET

**描述:** `/v4/finance/balance`

查询用户的理财总资产及持仓概况。

### 请求参数

无（通过请求头鉴权获取用户信息）

### 请求示例

<CodeBlock language="bash" title="Request">
{`curl -G "https://sapi.xt.com/v4/finance/balance" \\
  -H "validate-appkey: $APPKEY" \\
  -H "validate-timestamp: $TIMESTAMP" \\
  -H "validate-signature: $SIGNATURE" \\
  -H "Content-Type: application/json"
`}
</CodeBlock>

### 响应示例

<CodeBlock language="json" title="Response">
{`{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": {
    "totalAssets": "10500.00",
    "totalAssetsBtc": "0.24",
    "lastInterestAmount": "12.34",
    "lastInterestAmountBtc": "0.00028",
    "totalInterestAmount": "123.45",
    "totalInterestAmountBtc": "0.0028"
  }
}`}
</CodeBlock>

### 响应字段

| 字段 | 类型 | 描述 |
| --- | --- | --- |
| totalAssets | string | 总资产（USDT估值） |
| totalAssetsBtc | string | 总资产（BTC估值） |
| lastInterestAmount | string | 昨日收益 |
| lastInterestAmountBtc | string | 昨日收益（BTC） |
| totalInterestAmount | string | 累计收益 |
| totalInterestAmountBtc | string | 累计收益（BTC） |
