**请求类型** GET

**接口地址:** `/future/user/v1/position/adl`

---

### 限流规则

200次/秒/apikey

---

### 请求参数

| 参数名   | 类型   | 是否必填 | 默认值 | 描述 |
| -------- | ------ | -------- | ------ | ---- |
| symbol   | string | 否       | N/A    | 交易对（不传时查询所有交易对的持仓信息） |

---

### 请求示例
```bash
    curl -G "https://fapi.xt.com/future/user/v1/position/adl" \ 
    -H "validate-appkey: $APPKEY" \  # 必须
    -H "validate-timestamp: $TIMESTAMP" \  # 必须
    -H "validate-singature: $SINGATURE" \  # 必须
    -H "Content-Type: application/x-www-form-urlencoded" \  # 必须
```

---

### 响应示例

```json title="Response"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": [
    {
      "longQuantile": 0,   // 多头仓位ADL档位
      "shortQuantile": 0,  // 空头仓位ADL档位
      "symbol": ""         // 交易对
    }
  ],
  "returnCode": 0
}
```

---

### 错误码

| **Error code**               | **Desc**                       |
| ---------------------------- | ------------------------------ |
| invalid_symbol               | 交易对不存在                         |
