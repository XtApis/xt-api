**类型:** GET

**描述:** `/v4/referal/invite/user/balance`

### 参数

| 名称 | 类型   | 是否必需 | 默认值 | 描述 | 取值范围 |
| ---- | ------ | --------- | ------- | ----------- | ------ |
| uid  | number | true      |         | 用户ID     |        |

---

### 响应示例

```json title="响应"
{
  "ma": [],
  "mc": "SUCCESS",
  "rc": 0,
  "result": {
    "userId": 123456789,
    "role": "PARTNER",
    "balance": 123456
  }
}
```
