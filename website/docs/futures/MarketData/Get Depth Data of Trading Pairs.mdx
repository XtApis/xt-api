**Type:** GET

**Description:** `/future/market/v1/public/q/depth`

#### **Limit Flow Rules**

10/s/ip

<br />> Noteï¼šThis method does not require a signature.

### Parameters

| Name   | Type    | Mandatory | Default | Description           | Ranges |
| ------ | ------- | --------- | ------- | --------------------- | ------ |
| symbol | string  | true      | N/A     | Trading pair          |        |
| level  | integer | true      | N/A     | Level (min:1, max:50) |        |

### Request Example

```bash title="Request"
{
  curl -G "https://fapi.xt.com/future/market/v1/public/q/depth"  \
      -d "symbol=btc_usdt" \ # required
      -d "level=1" \ # required
}
```

### Error Code

| Error Code | Desc |
|-------------|----------------|
| invalid_symbol | Symbol does not exist |
| invalid_level | The minimum value of level is 1 and must be an integer |
