**Type:** GET

**Description:** `/future/market/v1/public/q/kline`

#### Noteï¼š

This method does not require a signature.

### Limit flow rules

None

### Parameters

| Name | Type | Mandatory | Default | Description | Ranges |
| --- | --- | --- | --- | --- | --- |
| symbol | string | true | N/A | Trading pair |  |
| interval | string | true | N/A | Time-interval | 1m;5m;15m;30m;1h;4h;1d;1w |
| startTime | integer | false | N/A | Start time |  |
| endTime | integer | false | N/A | End time |  |
| limit | integer | false | N/A | Limit |  |

### Request Example

```bash title="Request"
{
  curl -G "https://fapi.xt.com/future/market/v1/public/q/kline" \
      -d "symbol=eth_usdt" \ # required
      -d "interval=1m"  # required 
}
```

### Error Code

| Error Code | Desc |
|-------------|----------------|
| invalid_symbol | Symbol does not exist |
| invalid_limit | The minimum value of limit is 1, the maximum is 1500, and it must be an integer |
| invalid_interval | Invalid time interval |
| invalid_time | Invalid start or end time |
