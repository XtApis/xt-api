### Request Examples

```bash title="Request"
  curl --location --request GET 'https://sapi.xt.com/v4/public/time' \
    --header 'accept: */*' \
    --header 'Content-Type: application/json' \
```

Response Example

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": {
    "serverTime": 1662435658062
  }
}
```
