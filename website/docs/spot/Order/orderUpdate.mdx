**Type:** PUT **Description:** `/v4/order/{orderId}`

### Parameters

| name     | type   | mandatory | default | description | ranges |
| -------- | ------ | --------- | ------- | ----------- | ------ |
| orderId  | number | true      |         | order ID    |        |
| price    | number | true      |         | Price       |        |
| quantity | number | true      |         | Quantity    |        |

#### **Limit Flow Rules**

50/s/apikey

### Parameters Example

```
curl --location --request PUT 'https://sapi.xt.com/v4/order/xxxxxx' \
--header 'accept: */*' \
--header 'Content-Type: application/json' \
--header 'validate-algorithms: HmacSHA256' \
--header 'validate-recvwindow: 60000' \
--header 'validate-appkey: xxxxxxxxxx' \
--header 'validate-timestamp: xxxxxxxxxx' \
--header 'validate-signature: xxxxxxxxxx' \
--data '{
    "orderId": "xxxxxx",
    "price": 10,
    "quantity": 2
}'
```
