### Description

**Type:** POST

**Description** `/future/trade/v1/order/list-open-order`

---

### Limit Rule

- 10 次/秒/apikey

---

### Parameters

| Name | Type | Mandatory | Default | Description | Ranges |
| --- | --- | --- | --- | --- | --- |
| symbol | string | No | N/A | 交易对名称 | 例如：`eth_usdt`, `btc_usdt` |

---

### Request Example

<CodeBlock language="bash" title="Request">
{`curl -X POST "https://fapi.xt.com/future/trade/v1/order/list-open-order" \
  -H "validate-appkey: $APPKEY" \  # Required
  -H "validate-timestamp: $TIMESTAMP" \  # Required
  -H "validate-singature: $SINGATURE" \  # Required
  -H "Content-Type: application/x-www-form-urlencoded" \  # Required
  -d "symbol=btc_usdt"  # Required
`}
</CodeBlock>
