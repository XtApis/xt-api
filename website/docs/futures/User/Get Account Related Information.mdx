### Description

**Type** GET

**Description:**`/future/user/v1/account/info`

---

### Limit Rule

- 200/s/apikey

---

### Parameters

_None_

---

### Request Example

```bash title="Request"
curl -G "https://fapi.xt.com/future/user/v1/account/info" \
  -H "validate-appkey: $APPKEY" \        # Required
  -H "validate-timestamp: $TIMESTAMP" \  # Required
  -H "validate-singature: $SINGATURE" \  # Required
  -H "Content-Type: application/x-www-form-urlencoded"   # Required
```

### Response Example

```json title="Response"
{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "accountId": 0, // Account ID
    "allowOpenPosition": false, // Is it possible to open position
    "allowTrade": false, // Is it possible to trade
    "allowTransfer": false, // Is it possible to transfer
    "openTime": "", // Opening time
    "state": 0, // User status
    "userId": 0 // User ID
  },
  "returnCode": 0
}
```
