### 请求格式

```
trade@{symbol}
```

- 示例：`trade@btc_usdt`
- 推送频率：实时

---

### 请求示例

**订阅** — 建立 WebSocket 连接后，向服务器发送下列 JSON 字符串。

```json title="SUBSCRIBE"
{
  "method": "SUBSCRIBE",
  "params": [
    "trade@btc_usdt"
  ],
  "id": "test"     // 此参数可自定义
}
```

---

### 响应示例

```json
{
  "topic": "trade",
  "event": "trade@btc_usdt",
  "data": {
    "s": "btc_index", // 交易对
    "p": "50000",     // 价格
    "a": "0.1",       // 数量
    "m": "BID",       // 成交方向：BID = 买入，ASK = 卖出
    "t": 123124124     // 时间戳
  }
}
```
