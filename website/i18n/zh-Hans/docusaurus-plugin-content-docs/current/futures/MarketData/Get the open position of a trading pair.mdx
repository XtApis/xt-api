**类型:** GET

**描述:** `/future/market/v1/public/contract/open-interest`

#### **限流规则**

1次/秒/IP

### 注意：

此方法不需要签名。

### 参数

| 名称   | 类型   | 是否必需 | 默认值 | 描述     | 范围 |
| ------ | ------ | -------- | ------ | -------- | ---- |
| symbol | string | 是       | N/A    | 交易对   |      |

### 响应示例

import CodeBlock from '@theme/CodeBlock';

<CodeBlock language="json" title="Response">
  {`{
  "error": {
    "code": "",
    "msg": ""
  },
  "msgInfo": "",
  "result": {
    "symbol": "",          // 交易对
    "openInterest": "",    // 持仓量
    "openInterestUsd": 0,  // 持仓价值
    "time": ""             // 时间
  },
  "returnCode": 0
}`}
</CodeBlock>
