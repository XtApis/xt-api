**类型:** GET

**描述:** `/future/market/v1/public/contract/open-interest`

#### **限流规则**

1次/秒/IP

### 注意：

此方法不需要签名。

### 参数

| 名称   | 类型   | 是否必需 | 默认值 | 描述     | 范围 |
| ------ | ------ | -------- | ------ | -------- | ---- |
| symbol | string | 是       | N/A    | 交易对   |      |

### 请求示例

```bash title="Request"
{
  curl -G "https://fapi.xt.com/future/market/v1/public/contract/open-interest" \
      -H "validate-appkey: $APPKEY" \  # 必须
      -H "validate-timestamp: $TIMESTAMP" \  # 必须
      -H "validate-singature: $SINGATURE" \  # 必须
      -H "Content-Type: application/x-www-form-urlencoded" \  # 必须
      -d "symbol=eth_usdt"  # 必须
}
```

| 错误码 | 描述 |
|-------------|----------------|
