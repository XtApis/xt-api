### 请求
用户可以通过新的 WebSocket 订阅实时公告，帮助您在最新动态、活动信息和重要新闻发布时及时获取。

格式: `com_announcement_cn`
示例: `com_announcement_cn`

---

### 示例

#### Python
```python
from websocket import create_connection

ws = create_connection("wss://stream.xt.com/notice")

# 发送订阅请求
ws.send('{"topic": "com_announcement_cn", "event": "com_announcement_cn"}')
print("Subscribed to English announcement channel")
```
