**Type:** GET **Description:** `/future/trade/v1/entrust/plan-list-history`

---

### Limit Flow Rules

200/s/apikey

---

### Parameters

| Name | Type | Mandatory | Default | Description | Ranges |
| --- | --- | --- | --- | --- | --- |
| symbol | string | TRUE | N/A | Trading pair (if not provided, queries all pairs) | — |
| direction | string | FALSE | NEXT | Query direction | PREV; NEXT |
| id | integer | FALSE | N/A | Record ID | — |
| limit | integer | FALSE | 10 | Limit number of records | — |
| startTime | integer | FALSE | N/A | Start time | — |
| endTime | integer | FALSE | N/A | End time | — |

---

### Request Example

```bash
curl -G "https://fapi.xt.com/future/trade/v1/entrust/plan-list-history" \
    -H "validate-appkey: $APPKEY"              # Required
    -H "validate-timestamp: $TIMESTAMP"        # Required
    -H "validate-signature: $SIGNATURE"        # Required
    -H "Content-Type: application/x-www-form-urlencoded"  # Required
    -d "symbol=btc_usdt"                       # Required
    -d "direction=NEXT" \
    -d "limit=10"
```

---

```json
        "clientOrderId": "", // Client order ID

        "createdTime": 0, // Creation time
        "entrustId": 0, // Order ID

        "marketOrderLevel": 0, // Market order level

        "positionSide": "", // Position side
        "price": 0, // Entrust price
        "state": "", // Order status: NOT_TRIGGERED, TRIGGERING, TRIGGERED, USER_REVOCATION, PLATFORM_REVOCATION, EXPIRED

        "symbol": "", // Trading pair
        "timeInForce": "", // Time in force
        "triggerPriceType": "" // Trigger price type
      }
    ]
  },
  "returnCode": 0
}
```
