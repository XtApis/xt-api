部分接口设有流量控制（相应接口会说明其流量限制）。  
流量限制主要分为**网关流量限制**和 **WAF 流量限制**。

- 如果接口请求触发网关流量限制，将返回 **429**，表示访问频率超过限制，IP 或 `apiKey` 将被封锁。
- 网关流量限制分为**基于 IP** 和**基于 apiKey** 的流量限制。

**示例：**

- IP 流量限制：`100/s/ip` → 每个 IP 对该接口每秒最多 100 个请求。
- apiKey 流量限制：`50/s/apiKey` → 每个 apiKey 对该接口每秒最多 50 个请求。
