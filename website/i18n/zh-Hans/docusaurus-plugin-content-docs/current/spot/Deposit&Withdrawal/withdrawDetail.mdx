**类型:** GET **描述:** `/v4/withdraw`

### 参数

| 名称 | 类型 | 是否必填 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| recordId | string | 否 | N/A | 提币记录ID，从**提币**接口获得。建议优先使用此参数。 |  |
| clientOrderId | string | 否 | N/A | 自定义客户端ID |  |

### 注意事项

- 限流规则：**每个apikey每秒1次请求**。
- 如有 `recordId` 请优先使用，否则使用 `clientOrderId`。
- 此接口用于获取特定提币记录的详细信息。

### 请求示例

```json title="Request"
  curl --location --request GET 'https://sapi.xt.com/v4/withdraw?recordId=xxxxxxxxx' \
    --header 'accept: */*' \
    --header 'Content-Type: application/json' \
    --header 'validate-algorithms: HmacSHA256' \
    --header 'validate-recvwindow: 60000' \
    --header 'validate-appkey: xxxxxxxxxx' \
    --header 'validate-timestamp: xxxxxxxxxx' \
    --header 'validate-signature: xxxxxxxxxx' \
```    

### 响应示例

```json title="响应"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": {
    "id": 100, // 提币记录ID
    "clientOrderId": 10, // 客户端ID
    "type": 0, // 类型：CHAIN_TRANSFER - 区块链提币，INTERNAL_TRANSFER - 内部提币
    "currency": "btc", // 币种
    "address": "xxxxx", // 提币地址
    "status": "REVIEW", // 提币状态（参见充提币状态参考）
    "amount": 0.1, // 提币数量
    "fee": 0.0001, // 提币手续费
    "chain": "Tron", // 转账网络
    "memo": "yyyyy", // Memo/Tag（如适用）
    "confirmations": 2, // 区块确认数
    "transactionId": "490267492", // 交易哈希
    "createdTime": 1737093343000 // 提币申请时间（毫秒）
  }
}
```
