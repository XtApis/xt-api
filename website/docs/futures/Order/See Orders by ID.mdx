**Type:** GET

**Description:** `/future/trade/v1/order/detail`

### Parameters

| Name    | Type    | Mandatory | Default | Description | Ranges |
| ------- | ------- | --------- | ------- | ----------- | ------ |
| orderId | integer | true      | N/A     | Order ID    |        |

### Limit Flow Rules

200/s/apikey

### Request Example

<CodeBlock language="bash" title="Request Example">
{`curl -G "https://fapi.xt.com/future/trade/v1/order/detail" \\
  -H "validate-appkey: $APPKEY" \\  # Required
  -H "validate-timestamp: $TIMESTAMP" \\  # Required
  -H "validate-singature: $SINGATURE" \\  # Required
  -H "Content-Type: application/x-www-form-urlencoded" \\  # Required
  -d "orderId=1234567890"  # Required
`}
</CodeBlock>
