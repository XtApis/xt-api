**请求类型** GET

**接口地址:** `/future/user/v1/compat/balance/list`

---

#### **限流规则**

200次/秒/apikey

---

### 请求参数

| 参数名         | 类型   | 是否必填 | 默认值 | 描述     |
| -------------- | ------ | -------- | ------ | -------- |
| queryAccountId | string | 否       | N/A    | 账户 ID  |

---

### 请求示例

```bash title="Request"
curl -G "https://fapi.xt.com/future/user/v1/compat/balance/list" \ 
  -H "validate-appkey: $APPKEY" \  # 必须
  -H "validate-timestamp: $TIMESTAMP" \  # 必须
  -H "validate-singature: $SINGATURE" \  # 必须
  -H "Content-Type: application/x-www-form-urlencoded" \  # 必须
  -d "queryAccountId=1234567890"
```
