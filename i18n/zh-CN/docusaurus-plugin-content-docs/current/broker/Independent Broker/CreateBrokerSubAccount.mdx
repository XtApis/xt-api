**类型:** POST

**描述:** `/v4/user/account`

### 参数

| 名称        | 类型    | 是否必填 | 默认值 | 描述                                    | 取值范围 |
| ----------- | ------- | -------- | ------ | --------------------------------------- | -------- |
| accountName | string  | 是       | -      | 子账户名称                              | 6-16     |
| type        | integer | 否       | 1      | 账户类型：1-普通账户，7-经纪商子账号    | 1, 7     |

---

### 响应示例

```json title="Response"
{
  "rc": 0,
  "mc": "SUCCESS",
  "ma": [],
  "result": true //true: 成功; false/null: 失败
}
```
