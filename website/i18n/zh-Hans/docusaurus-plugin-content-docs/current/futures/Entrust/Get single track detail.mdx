**请求类型:** GET
**描述:** `/future/trade/v1/entrust/track-detail`

---

### 限流规则

200/s/apikey

---

### 参数说明
| 名称 | 类型 | 必填 | 默认值 | 描述 | 范围 |
|------|------|------|--------|------|------|
| trackId | integer | TRUE | N/A | 跟踪单ID | — |

---

### 请求示例
```bash
curl -G "https://fapi.xt.com/future/trade/v1/entrust/track-detail" \
    -H "validate-appkey: $APPKEY"              # 必须
    -H "validate-timestamp: $TIMESTAMP"        # 必须
    -H "validate-signature: $SIGNATURE"        # 必须
    -H "Content-Type: application/x-www-form-urlencoded"  # 必须
    -d "trackId=1234567890"                    # 必须
```

---

### 错误码
| 错误码 | 描述 |
|---------|------|
| invalid_track_Id | 跟踪单不存在 |
