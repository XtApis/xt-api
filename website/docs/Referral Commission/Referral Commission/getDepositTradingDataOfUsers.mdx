**Type:** GET **Description:** `/v4/referal/invite/user/data`

### Parameters

| Name      | Type   | Mandatory | Default | Description     | Ranges |
| --------- | ------ | --------- | ------- | --------------- | ------ |
| startTime | number | false     |         | Start time      |        |
| endTime   | number | false     |         | End time        |        |
| uid       | number | true      |         | Queried user ID |        |
