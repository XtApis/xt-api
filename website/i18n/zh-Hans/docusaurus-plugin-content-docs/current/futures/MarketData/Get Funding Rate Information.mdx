**类型:** GET

**描述:** `/future/market/v1/public/q/funding-rate`

#### **限流规则**

1次/秒/IP

### 注意：

此方法不需要签名。

### 参数

| 名称   | 类型   | 是否必需 | 默认值 | 描述     | 范围 |
| ------ | ------ | -------- | ------ | -------- | ---- |
| symbol | string | 是       | N/A    | 交易对   |      |
