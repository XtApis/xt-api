**Type:** GET

**Description:** `/future/market/v1/public/contract/risk-balance`

#### Noteï¼š

This method does not require a signature.

### Limit flow rules

None

### Parameters

| Name | Type | Mandatory | Default | Description | Ranges |
| --- | --- | --- | --- | --- | --- |
| symbol | string | true |  | Trading pair |  |
| direction | string | false | NEXT | Direction (PREV: Previous page; NEXT: Next page) | PREV;NEXT |
| id | integer | false | N/A | id |  |
| limit | integer | false | 10 | Limit |  |

### Request Example

```bash title="Request"
{
  curl -G "https://fapi.xt.com/future/market/v1/public/contract/risk-balance" \
      -d "symbol=eth_usdt" \ # required
      -d "limit=10"
}
```
