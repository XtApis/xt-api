**类型:** GET

**描述:** `/v4/referal/invite/agent/rebate/data`

### 参数

| 名称 | 类型 | 是否必需 | 默认值 | 描述 | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| startTime | number | false |  | 佣金充值开始时间 |  |
| endTime | number | false |  | 佣金充值结束时间 |  |
| inviteCode | string | false |  | 注册邀请码 |  |
| type | number | true |  | 佣金类型 | 1-现货, 2-合约 |
| fromId | number | false |  | 起始ID |  |
| direction | string | false | NEXT | 查询方向 | PREV, NEXT |
| limit | int | false | 10 | 数量限制，最大100 | `1 <= limit <= 100` |

---

#### **备注**

仅支持查询过去三个月的数据。

---
